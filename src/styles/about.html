About

Easy Music is a website that helps you discover songs, albums, artists, and create playlists using the Spotify Web API.

  • Home:
    • Search through a variety of songs.
    • Songs display title, artist, album, and length.
    • After songs are searched, user can choose to add from the list of songs to their playlist.
    • Once songs are added, they can be removed using the red button.
    • Playlists will require a name to be saved locally.
  • Playlists:
    • Allows viewing of all saved playlists.
    • Once a playlist is saved, users can export them to their device.
    • User can load and edit existing playlists by clicking the load button.
    • Playlists can be deleted as needed.
    • Import playlists from JSON files to be edited or viewed.
  • About:
    • Gives information about the website as a whole.
    • Helps user understand how to use website.
